# Project Images Guide

## How to Add Project Images

### Location
Place all project images in the `public/projects/` directory.

### Required Images

You need to add 9 images total (3 for each project):

#### 1. AppleMusic Downloader
- `public/projects/applemusic1.png` - Main screenshot
- `public/projects/applemusic2.png` - Feature showcase
- `public/projects/applemusic3.png` - UI detail

#### 2. HackPod
- `public/projects/hackpod1.png` - Device photo
- `public/projects/hackpod2.png` - Hardware detail
- `public/projects/hackpod3.png` - Usage scenario

#### 3. Authenticator
- `public/projects/authenticator1.png` - CLI interface
- `public/projects/authenticator2.png` - TUI dashboard
- `public/projects/authenticator3.png` - Feature demo

### Image Specifications

**Format:** PNG or JPG
**Recommended Size:** 1920x1080px (16:9 ratio)
**File Size:** Keep under 500KB per image for optimal loading
**Quality:** High resolution, clear and professional

### Tips

1. Use consistent lighting and backgrounds
2. Show different aspects of each project
3. Include both code/UI screenshots and final product photos
4. For hardware projects (HackPod), include physical device photos
5. For software projects, show the actual application interface

### Alternative Approach

If you don't have screenshots yet, you can:
1. Use placeholder images temporarily
2. Take screenshots using your OS screenshot tool
3. Use online mockup generators for professional presentation
4. Export from Figma/design tools if you have mockups
